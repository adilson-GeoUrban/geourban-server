🌍 Privacy Policy - GeoUrban

1. Data Collection

GeoUrban collects only the data necessary to provide urban analytics services, including user inputs, technical data (IP address, browser), and interaction logs.

2. Data Usage

Data is used to generate AI-based urban analysis, improve platform performance, ensure security, and prevent fraud.

3. Data Sharing

We do not sell or share personal data for advertising purposes. Data may be shared only when required for legal compliance or technical operation.

🌐 International Compliance

GeoUrban complies with international data protection regulations, including:

  • LGPD (Brazil - Law 13.709/2018)
  • GDPR (European Union)

By using the platform, users agree to the processing of their data under these regulations.

🔐 Data Security

We implement security measures to protect user data against unauthorized access, loss, or misuse.
